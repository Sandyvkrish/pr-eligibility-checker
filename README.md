This is a simple web tool that gives users a quick idea of their eligibility for Canada PR based on a few basic details.

It’s built using plain HTML, CSS, and JavaScript, and can be easily added to any landing page or CMS like Elementor.
The user selects:
Age range
Education level
English score
Then clicks a button, and the tool gives a basic result like:
High chance
Medium chance
Low chance
It’s not an official system — just a simple logic-based checker for demo/landing page use.

Technologies used:
HTML
CSS
JavaScript
Netlify (for hosting)
